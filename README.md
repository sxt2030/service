Тут лежит код для автоматизации деплоя приложения на Go на инфраструктуру, которую подняли из соседнего репозитория infra.

После пуша в ветку uat, приложение автоматически тестируется автотестами, запихивается в контейнер и деплоится на стейдж сервер. После мержа в main можно вручную запустить задачу деплоя на прод.
Для этого используется тот же контейнер, что был проверен и создан в ветке uat. Все тестировалось на собственном гитлабе, поднятом на ВМ в Google Cloud.
